🎮 Tic Tac Toe (C++ Console Game)

A simple yet smart Tic Tac Toe game in C++ where you play against the computer.
Built using Object-Oriented Programming concepts with a basic AI opponent.

---

🚀 Features

- 🧠 Computer AI (tries to win + blocks your moves)
- 🎯 Clean 3x3 board display
- 📌 Position guide for easy input
- ⚠️ Input validation (handles invalid entries)
- 🔁 Turn-based gameplay (Player vs Computer)

---

🕹️ How to Play

- You are X
- Computer is O
- Enter positions from 1 to 9 based on the grid:

 1 | 2 | 3
---+---+---
 4 | 5 | 6
---+---+---
 7 | 8 | 9

- First to get 3 in a row (horizontal, vertical, diagonal) wins.

---

🛠️ How to Run

1. Clone the repository

git clone https://github.com/Vedansh-Choubey/CPP_Basics.git

2. Navigate to project folder

cd CPP_Basics

3. Compile the code

g++ tic_tac_toe.cpp -o game

4. Run the game

./game

---

🧠 AI Logic (Simple Strategy)

The computer follows these steps:

1. Win if possible
2. Block opponent if they are about to win
3. Take center position
4. Take corners
5. Pick any available space
